## Curation philosophy

Every resource in this library earned its place. The filtering criteria:

1. **Actually used** — not just bookmarked. Each resource was used at some point in building xiaoyanghu.com or in daily design work.
2. **Has a specific use case** — not just "it's good." There's a reason and a moment for each one.
3. **Not obvious** — if it's on every "top design resources" list already, it's probably not here. The point is to surface things you might not find on your own.

Items marked with ★ are tier 1 — Elena's personal must-sees. Start with these.

## Guidance for common questions

**"I don't know what style I want for my portfolio."**
Don't force it early. Start by looking at 20-30 portfolios and notice what you keep coming back to. Explore Pinterest for loose visual vibes. Over time, patterns emerge. Elena explored New Brutalism, nostalgic/retro, and minimal before her direction became clear — it took about 5 months.

**"I want to build a Framer portfolio from scratch."**
Start with Framer University. Then look at Remix Lab and framer.today for components. For visual inspiration, study alexyc.com and Arshad Riyas for minimal, or Marco Cornacchia for bento layout.

**"I'm vibe coding and need UI components."**
Start with React Bits and Magic UI for motion-heavy components. Fancy Components for creative effects. FreeFrontend for code snippets you can drop straight in. Uiverse for pure CSS elements.

**"Who should I follow on X for design inspiration?"**
Start with @samdape, @raunofreiberg, @ryolu_, and @ridd_design. These four cover different aesthetics and will give you a good spread of daily inspiration.

**"I want to improve my motion and animation skills."**
Follow @emilkowalski on X, study GSAP's docs and demos, and look at Mo's UX Portfolio (tsui-mou.com) for rich motion examples. Thibaut Crépelle (thibaut.cool) is also a strong reference for 3D and motion.
