## Design Inspiration

**★ Must see:**
- **Design Spells** (designspells.com) — a library of design details that feel like magic moments. Open this when you need a spark.

**Worth exploring:**
- **Rauno Freiberg** (rauno.me) — personal site of a Vercel design engineer. The attention to detail will make you want to rebuild your own site.
- **Things Inc.** (things.inc) — the Things app website. Study every hover, every transition.
- **awwwards** (awwwards.com) — daily picks of the most worth-seeing web design work.
- **FREAK MAG** (daytona-park.com/freakmag) — experimental visual design magazine.
- **Making Software** (makingsoftware.com) — principles and methods behind great software design.
- **GSAP** (gsap.com) — the animation library. Not just a tool, the demos are full of interaction inspiration.
- **Landingfolio** (landingfolio.com) — best landing page inspiration collection.
- **Spline** (spline.design) — 3D design tool in the browser.
- **Interface Craft** (interfacecraft.dev) — reference for interface interaction details.
- **Unicorn Studio** (unicorn.studio) — no-code WebGL tool.
- **Acctual** (acctual.com) — a crypto invoicing startup. One of Elena's biggest visual references. Sometimes the best inspiration comes from completely unexpected places.

## UI Component Libraries

Mostly React ecosystem. For web coding and vibe coding.

**★ Must see:**
- **React Bits** (reactbits.dev) — animated UI components for React. Makes your portfolio look like it took 3x longer to build.
- **FreeFrontend** (freefrontend.com) — free front-end code snippets. Huge range, easy to browse.

**Worth exploring:**
- **Fancy Components** (fancycomponents.dev) — creative components you won't find anywhere else.
- **Magic UI** (magicui.design) — beautiful motion components, great for landing pages.
- **Aceternity UI** (ui.aceternity.com) — high-quality React component collection.
- **Uiverse** (uiverse.io) — largest library of open-source UI elements, pure CSS.
- **Neobrutalism** (neobrutalism.dev) — neo-brutalist style components.
- **ui-layouts** (ui-layouts.com) — UI layout references with code.
- **21st.dev** (21st.dev) — AI-assisted front-end component creation.
- **CSS Buttons** (cssbuttons.io) — huge collection of CSS button styles.

## Framer Resources

**★ Must see:**
- **Framer University** (framer.university) — the go-to learning hub for everything Framer. Start here before anything else.

**Worth exploring:**
- **Remix Lab** (remixlab.framer.wiki) — Framer component wiki and resources.
- **framer.today** (framer.today/resources) — Framer resource aggregator.
- **Cursors** (cursors.framer.wiki) — cursor animation components. Small detail, big impact on feel.
- **Segment UI** (segmentui.com) — free components and UI kits for Framer & Figma.

## Portfolio Inspiration

Mostly entry-level and mid-level. Focus on visual style, information structure, and interaction details.

**★ Must see:**
- **alexyc** (alexyc.com) — one of the best minimal portfolios out there. Study the whitespace.
- **Arshad Riyas** (arshadriyas.framer.ai) — one of the best minimal portfolios out there.
- **Elisha Jeon** (elishajeon.com) — a completely new way to browse case studies. The interaction pattern is something you've never seen before.
- **Serena Li** (serenali.ca) — hard to imitate. The personal style is too strong.
- **Alric Monteiro** (alric.framer.website) — feels like stepping into a fully built cyberpunk universe.
- **Jackie Zhang** (jackiezhang.co.za) — strong visual and creative direction.
- **Xiaoyang Hu** (xiaoyanghu.com) — the portfolio this resource library was built from.

**Worth exploring:**
- **Ryo Lu** (ryo.lu) — the reference for OS-style portfolios. Everyone copies this. See the original.
- **Marco Cornacchia** (marco.fyi) — one of the best bento-style portfolios.
- **Emmi Wu** (emmi.framer.website) — visual direction makes you want to keep scrolling.
- **Liandong Zhou** (liandongzhou.com) — best neo-brutalist portfolio out there.
- **Thibaut Crépelle** (thibaut.cool) — 3D and motion work is seriously impressive.
- **Uttkarsh Mandaknale** (uttkarsh.design) — love the dreamy aesthetic.
- **Angelo Arcilla** (angeloarcilla.com) — no idea how long this took to build.
- **Oğuz Yağız Kara** (oguzyagiz.com) — when you're good enough that you don't need a portfolio, this is how you make one anyway.
- **tom things** (tomthings.fr) — design that feels alive.
- **adam syed** (syedadam.com) — look at this when you have a lot of projects to show.
- **Mayank Kinger** (kingermayank.com) — solid. Just solid.
